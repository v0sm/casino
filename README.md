# Goose Casino

Симуляция казино с игроками, гусями и фишками.

Проект реализует небольшую модель казино, где:
- игроки делают ставки, выигрывают и проигрывают;
- гуси воруют деньги или гогочут, влияя на баланс игроков;
- расчеты ведутся через объекты фишек с разными номиналами.

## Возможности

- Регистрация новых игроков с начальным балансом.
- Регистрация гусей двух типов:
  - WarGoose — ворует деньги у игроков.
  - HonkGoose — гогочет и изменяет баланс игрока.
- Моделирование событий казино:
  - атака WarGoose;
  - гогот HonkGoose;
  - выигрыш игрока;
  - проигрыш игрока;
  - ставка;
  - обмен денег на фишки.
- Пошаговая симуляция с возможностью задать число шагов и seed для воспроизводимости.

## Установка и запуск

### Требования

- Python 3.10 или новее.
- Виртуальное окружение рекомендуется, но не обязательно.

### Установка
```
git clone https://github.com/v0sm/casino.git
cd casino

python -m venv venv
venv\Scripts\activate # Windows

source venv/bin/activate # Linux / macOS
pip install -r requirements.txt
```
### Запуск симуляции

По умолчанию симуляция запускается из `main.py`.
```
python src/main.py
```
## Основные сущности

- `Player`
  - имя;
  - текущий баланс;
  - коллекция фишек;
  - методы изменения баланса и ставок.

- `Goose`, `WarGoose`, `HonkGoose`
  - базовый гусь и два специализированных типа;
  - WarGoose ворует деньги у игроков;
  - HonkGoose гогочет и изменяет баланс игрока на заданную величину.

- `Chip`
  - объект фишки с номиналом;
  - статический набор доступных номиналов.

- `Casino`
  - хранит коллекции игроков, гусей и банка фишек;
  - выбирает и выполняет случайные события;
  - печатает ход симуляции и итоговую статистику.
